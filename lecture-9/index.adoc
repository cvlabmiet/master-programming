=== Характеристика библиотеки

* Boost -- набор библиотек, по большей части "in-header"
* Распространяется по собственной лицензии https://www.boost.org/users/license.html[Boost Software License]
* Основной сайт: https://www.boost.org
* Время возникновения -- 1999 г.
* Разработка ведётся на https://github.com/boostorg/boost[github] (все библиотеки подключаются в виде git-модулей)
* Система сборки своя, boostjam
* Большинство идей и библиотек попали в STL
